이미지 태그

img 태그는 이미지를 연결시켜 문서에 보이도록 해주는 태그입니다.

img 태그는 닫는 태그 없습니다.

링크를 연결할때는 href 속성을 쓰고, img 태그에 사용하는 속성은 src입니다.

alt라는 속성으로 사용된 이미지에 대한 설명을 입력합니다.

위에서 사용된 alt 속성은 이미지가 깨지거나, 액박이 뜨더라도 텍스트로 표현이 됩니다.

이미지가 잘나온다면 보이지 않습니다.

img 태그는 인라인 요소입니다.





작성중인 html 파일과 동일한 위치에 이미지를 불러올 경우 : ./파일이름.확장자


작성중인 html 파일과 동일한 위치에 폴더가 있고, 그 안에 있는 이미지를 불러올 경우 : ./폴더이름/파일이름.확장자



상위 폴더 ../

너비 : width 높이 : height

너비값 높이값 둘중에 하나만 조정을 하면 나머지 크기는 정비율에 맞춰서 자동으로 조정된다.
