table

  1. table 태그는 표를 만들때 사용하는 태그
  2. 옛날 홈페이지를 제작하는 방식인데, 수정이 어렵고, 잘 일그러져서 조금씩 수정하고 확인하고를 반복해야 한다.
  3. 행(row)과 열(column)로 구성한다. 오-가로와 열-세로, 행-가로 종-세로.
  4. table -> tr -> th/td의 순서로 마크업한다.
  5. tr, td, th는 모두 자식요소로 들어오는 태그들이다.
  6. table에 속성값을 추가해서 선을 만들고 (border="선굵기"), 선의 간격(cellspacing="간격수치")과 셀의 여백(cellpadding="여백수치"), 표의 너비(width="너비값"), 표의 높이(hegint="높이값")을 조절 할 수 있다.
  7. tr 태그에 align 속성으로 내용의 글자를 정렬한다.
  8. tr : 행(row)을 정의할 때 사용하는 태그이다.
  9. td : 열(column)을 정의할 때 사용하는 태그이다.
  10. th : td와 같은 기능인데, 제목역할을 하는 셀을 정의할 때 사용하는 태그이다.
