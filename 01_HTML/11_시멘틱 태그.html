시멘틱 태그

div 태그 대신에 의미를 전달 할 수 있는 태그

시멘틱 태그들은 기본적으로 블락 요소

텍스트, 인라인, 블락 모두 포함가능한 요소

  1. header : HTML 문서의 헤더 영역을 의미하는 태그(제목, 로고, 네비게이션, 검색창 등이 들어가는 자리)

    텍스트, 인라인, 블락 모두 포함가능. div와 비슷한 개념으로 이해하면 편하다.

    header 태그가 또 들어오거나, footer 태그가 들어가면 안된다. 작동은 o

    헤더 태그 사용법

  2. main : HTML 문서의 주요 내용이 들어오는 태그

    텍스트, 인라인, 블락 모두 포함 가능

    메인 태그 사용법

  3. footer : HTML 문서의 푸터 영역을 의미하는 태그. 저작권 정보, 관련문서 링크, 패밀리 사이트, 주소, 영업시간, 사업자번호, 전화번호 등등등이 들어오는 영역

    텍스트, 인라인, 블락 모두 포함 가능

    header 태그나 footer 태그가 또 들어오면 안된다. 작동은 o

    푸터 태그 사용법

  4. aside

    메인 컨텐츠와 직접적으로 관련이 없는 영역을 의미하는 태그. 홈페이지 같은 곳에서 마우스로 스크롤해서 화면이 움직일 때에 같이 따라다니는 작은 창같은 것을 aside 태그에 많이 집어 넣는다.

    SNS, 최근 본 상품, 광고 배너, 화면 상단으로 이동 등등등

    보통은 HTML 오른쪽이나 왼쪽에 사이드 메뉴나 광고 등의 영역으로 사용

    aside 태그 사용법

  5. nav

    HTML 문서의 메인메뉴나 목차 등을 정의해주는 태그

    통상적으로 헤더 태그 내에서 많이 사용된다.

    nav 태그 사용법

  6. section

    HTML 문서 내에서 맥락이 같은 요소들을 주제별로 그룹화 해주는 태그

    텍스트, 인라인, 블락 요소 모두 포함가능

    통상적으로 메인 태그 안에서 많이 사용된다.

    section 태그 사용법

  7. article

    HTML 문서내에서 독립적으로 배포, 재사용이 가능한 게시물, 뉴스기사, 블로그 포스팅 등을 나타낼때 사용

    텍스트, 인라인, 블락 다 포함가능

    article 태그 사용법

  8. address

    footer 태그내에서 주소, 사업자등록번호, 사업장 이름, 전화번호 등을 나타낼때 사용. 반드시 footer 태그가 아니더라도 사용 할 수 있으나, 보통은 footer 태그내에서 사용한다.

    기본 스타일(p 태그 아래위로 여백이 좀 생기는 것과 같은 느낌을 기본 스타일)로 글자 기울어짐 효과가 들어가있다.

    address 태그 사용법
