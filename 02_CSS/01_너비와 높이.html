width : 요소의 너비를 나타내는 속성

  • 단위 : px, %, auto(정해준 너비를 없앨때 사용됨)
  • 300px
  • 100px
  • 50%

height : 요소의 높이

  • 단위 : px, %, auto(정해준 너비를 없앨때 사용됨)
  • 250*200
  • 200px

min-width : 요소의 최소 너비

단위 : px, %, auto(정해준 너비를 없앨때 사용됨)

최소 너비 : 800px

max-width : 요소의 최대 너비

단위 : px, %, auto(정해준 너비를 없앨때 사용됨)

최대너비 800px

min-height : 요소의 최소 높이

단위 : px, %, auto(정해준 높이를 없앨때 사용됨)

최소 높이 : 800px

max-height : 요소의 최대 높이

단위 : px, %, auto(정해준 높이를 없앨때 사용됨)

최대높이 800px
