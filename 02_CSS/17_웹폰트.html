link 태그로 연결해서 사용

  1. 구글폰트 검색 > 한국어
  2. 원하는 폰트 선택
  3. + select this style
  4. link 부분 복사 후 head 태그 안에 붙여넣기
  5. CSS rules to families 부분 복사해서 헤드>스타일 태그 안에 적용할 태그에 넣기



@import로 연결해서 사용

스타일시트에 사용(외부, 내부 둘다 가능)

  1. 구글 폰트 검색 > 한국어
  2. 원하는 폰트 선택
  3. + select this style
  4. @import 부분을 복사한 후 스타일시트 안에 붙여넣기
  5. CSS rules to families 부분 복사해서 적용할 태그에 넣기
