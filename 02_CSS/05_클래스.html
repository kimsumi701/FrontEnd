class

태그에 이름은 줄 때 사용, 여러개의 태그에 같은 이름을 사용할 수 있다. 띄워쓰기 하나로 하나의 태그에 여러개의 이름도 사용할 수 있다.

id

태그에 고유명칭을 줄 때 사용, 여러개의 태그에 같은 이름을 사용할 수 없다. 아이디와 클래스는 같은 이름으로 사용이 가능하다.

규칙

클래스나 아이디의 이름은 영어로 작성해야한다. 대/소문자를 구분한다. 시작자리에 들어갈 수 있는 기호는 없다. 기호는 영문을 사용한 다음 중간에 언더바나 물결 같은 것을 쓴다. 관례적으로 내용과 관련이 있는 영문 이름으로 사용해야 한다.

(내부 스타일 시트의 경우)해드태그의 타이틀 태그 밑에 스타일 태그 안에서 사용되어 진다. 중괄호 안에 들어갈 속성들은 작성한다.

아이디는 #으로 호출, 클래스는 .으로 호출한다. 나중에 적은 값이 우선순위로 적용된다. 클래스보다 아이디가 우선순위로 적용된다.




    클래스나 아이디의 이름을 작성하는 방식
  • sbsacademyfrontend : 예시로 사용될 이름
  • 카멜 케이스(Camel case) : SbsAcademyFrontEnd
  • 스네이크 케이스(Snake Case) : sbs_academy_front_end
