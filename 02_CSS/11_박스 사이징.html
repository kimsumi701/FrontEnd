box-sizing : 박스의 크기를 결정하는 방식

내가 지정한 높이와 너비 안에 padding과 border의 포함 여부를 결정하는 방식

속성값 : content-box, border-box

content-box : 지정한 요소의 크기에 padding, border를 미포함

border-box : 지정한 요소의 크기에 padding, border를 포함
