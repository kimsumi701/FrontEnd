문자관련 스타일

  1. font-size : 글자 크기 조정

    단위 : px, %, em, rem

    em은 부모요소의 사이즈를 기준으로 곱하기라고 생각하면 된다.

    휴대폰 UI/UX 기획자 휴대폰의 편리한 사용자 인터페이스를 제공하기위하여 사용자의 요구사항을 분석한다. 하드웨어 특성을 고려한 의 기능을 분석하여 UI/UX 디자인을 검토한다.

  2. font-family : 글자체 조정

    font-family : dotum; -> 영문 글자체 이름을 쓸 경우에는 속성값에 영문 글자체명을 적어주면 된다.

    font-family: '돋움'; -> 한글 글자체 이름을 쓸 경우에는 작은 따옴표 안에 한글 글자체명을 적어주면 된다.

    휴대폰 UI/UX 기획자 휴대폰의 편리한 사용자 인터페이스를 제공하기위하여 사용자의 요구사항을 분석한다. 하드웨어 특성을 고려한 의 기능을 분석하여 UI/UX 디자인을 검토한다.

  3. font-weight : 글자 두께 조정

    속성값 : normal, bold, bolder, lighter, 별도의 단위 사용 없이 숫자 100부터 100단위로 900까지 쓸 수 있다. 노말이 400정도, 그 이상은 볼드, 그 이하는 라이트라고 생각하면 된다.

    휴대폰 UI/UX 기획자 휴대폰의 편리한 사용자 인터페이스를 제공하기위하여 사용자의 요구사항을 분석한다. 하드웨어 특성을 고려한 의 기능을 분석하여 UI/UX 디자인을 검토한다.

  4. line-height : 행간(줄과 줄 사이 간격) 조정

    단위 : px, %, em

    글자 크기가 클수록 더 크게 줘야한다. 글자 크기보다 작으면 글자가 겹치는 현상이 일어난다.

    통상적으로 작성되는 본문의 행간은 1.4~1.6em 정도가 적절하다고 생각한다.

    줄이 한줄일 경우에는 행간 값이 하단 높이값으로 된다. 그래서 글자가 1줄만 있을때에는 높이값이 없이 line-height 값을 높이값으로 주면 가로 가운데 정렬이 이루어진다.

    휴대폰 UI/UX 기획자 휴대폰의 편리한 사용자 인터페이스를 제공하기위하여 사용자의 요구사항을 분석한다. 하드웨어 특성을 고려한 의 기능을 분석하여 UI/UX 디자인을 검토한다.

  5. color : 글자 색상을 조정

    색상명(red, green, blue 등등) 정확한 영문 색상명이 들어와야 한다. 색상코드(#000000, #ffffff 등)도 사용할 수 있다. 앞에 #을 꼭 붙여야한다. 6개의 숫자 또는 영문이 반복되는 경우에는 3개만 적어도 사용이 가능하다.(#000000=#000) rgb컬러(N, N, N, 0~1사이 소수점)(red 수치, green 수치, blue 수치, 오퍼시티)(수치에 들어가는 값은 0~255까지 가능)
