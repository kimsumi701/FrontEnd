함수

  • 여러 줄의 명령을 하나로 묶어주는 주머니의 역할을 한다.
  • 영어로는 function이라고 함.
  • 반복되는 명령을 주머니안에 넣어두고 필요할 때에 주머니의 이름만 불러내면 명령이 실행되도록 한다.
  • 함수를 만드는 방법(함수를 선언하는 방법)
    function 함수이름 () { 명령들을 한꺼번에 모아둔다 }
  • 함수를 선언만 하면 해당 명령이 실행되지 않는다.
  • 함수를 선언하고 명령이 실행되도록 하려면 함수를 호출해야 한다.
  • 함수를 호출하는 방법
    함수이름();
  • 호출을 먼저하고 뒤에 선언해도 상관 없다.
    호이스팅의 개념이 적용된다.
